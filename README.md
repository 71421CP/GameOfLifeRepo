# GameOfLifeRepo
For the Code Competition Game of Life 2018 by IT-Talents
Developed by Christian Pütz

Classic game of life. Cells live and die according to the number of neighbor cells.
Setting allow further customization of the game rules.

To play start the GameOfLife.exe in the folder GameOfLife 1.0.
All files and folders in this folder need to stay this way.
Startup the executable and enjoy :)
